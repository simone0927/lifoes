📖 Come configurare
  1. Crea un repository privato su GitHub
  2. Genera un Personal Access Token con scope repo
  3. Inserisci username, nome repository e token sopra
  4. Clicca Salva config → Test connessione
  5. Usa Push per salvare i dati su GitHub
  6. Usa Pull per ripristinare i dati su un nuovo dispositivo
